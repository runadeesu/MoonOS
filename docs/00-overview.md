# Moon OS 概要

Moon OS は、Windows・macOS・Linux の優れた設計思想を参考にしながら、独自の
アーキテクチャ・GUI・操作体験を持つ、ゼロから作る自由 OS です。

既存 OS のコードやアセットは一切使用しません。すべてのコンポーネントは
このリポジトリの中でオリジナルに実装し、標準規格（UEFI、GPT、TCP/IP、
POSIX 互換 API など）にのみ準拠します。

## 目標

- 初心者でも迷わない、上級者は自由に拡張できる
- 軽量・高速・省電力
- セキュリティ・バイ・デザイン（メモリ安全な言語を優先、最小権限）
- Moon Light デザインテーマによる一貫した美しい GUI
- ゲーム・クリエイティブ用途・開発用途のすべてに耐える性能
- 長期的にメンテナンスできるモジュール設計

## 開発言語の使い分け

| レイヤー | 言語 | 理由 |
|---|---|---|
| ブートローダー / カーネル | Rust (`no_std`) | メモリ安全性、ゼロコスト抽象化、C 並みの制御 |
| 低レベル割り込み・コンテキストスイッチ | Rust + インラインアセンブリ | 移植性を保ちつつ CPU 固有処理を隔離 |
| ドライバ | Rust | クラッシュしてもカーネル全体を落とさない設計にしやすい |
| GUI シェル / Compositor | Rust + C++（将来的に） | 描画パイプラインの性能とエコシステム活用 |
| システムユーティリティ / ツール | Rust, Python | 開発速度とメンテナンス性 |
| ビルド・テスト自動化 | Python, Bash, CMake | 既存エコシステムとの親和性 |

## 開発は「小さく完成させる」方式で進める

このリポジトリは、巨大な設計書を一度に書いて終わりにするのではなく、
**実際にビルドでき、QEMU で起動確認したコード**を各ステップごとに積み上げます。
各ステップの成果物は `docs/02-roadmap.md` にチェックリストとして記録します。
