# Moon OS 開発ロードマップ

大規模 OS 開発を「設計 → 実装 → ビルド → テスト → 修正」の小さいサイクルで
進めるためのチェックリストです。各ステップは単独でビルド・起動確認できる
状態まで完成させてから次に進みます。

## Phase 0 — 基盤 (Foundation)

- [x] **Step 1: 最小フリースタンディングカーネル**
  - `bootloader` crate 経由で UEFI / BIOS 両方から起動
  - フレームバッファへの描画（Moon Light テーマの背景色 + ビットマップフォント）
  - シリアルポート (COM1) へのデバッグログ出力
  - QEMU (BIOS: SeaBIOS, UEFI: OVMF) で起動確認済み

- [ ] **Step 3: メモリ管理**

## 進め方の原則

- 1 ステップ = 1 PR = 実際に `cargo build` が通り、QEMU で起動確認できる単位
- 大きな機能（例: ネットワークスタック全体）は着手前にサブステップへ分解する
- 各ステップ完了時にこのファイルのチェックボックスを更新する
