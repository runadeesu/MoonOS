# GUI構造（Moon Light デザインテーマ）

## デザインコンセプト

「夜空」「月」「透明感」をテーマに、ガラス調 (glassmorphism) の
UI を採用します。

| 要素 | 値 |
|---|---|
| 背景（ダーク基調） | `#0A0C18`（実装済み: `framebuffer.rs` の `BG`） |
| 前景テキスト | `#D6E0FF`（実装済み: `framebuffer.rs` の `FG`） |
| アクセント（月光ブルー） | `#7FB2FF`（予定） |
| アクセント（アクリル/グラス） | 半透明 + ガウシアンブラー（Compositor 実装後） |
| 角丸 | ウィンドウ 12px, ボタン 8px（予定） |
| アニメーション | 120–200ms, ease-out（予定） |

## レイヤー構成（計画）

```
┌───────────────────────────────────────────┐
│ Moon Shell                                  │
│ ├── ステータスバー（時計・通知・クイック設定）│
│ ├── ドック / タスクバー                     │
│ ├── ランチャー（アプリ検索・起動）           │
│ └── 通知センター                            │
├───────────────────────────────────────────┤
│ Window Manager（タイル/フロート両対応）      │
├───────────────────────────────────────────┤
│ Compositor（GPU合成、ブラー、透明度）         │
├───────────────────────────────────────────┤
│ Moon UI Toolkit（ウィジェット、テーマエンジン）│
├───────────────────────────────────────────┤
│ Graphics Backend（フレームバッファ→将来GPU）  │
└───────────────────────────────────────────┘
```

## 現状の実装（Step 1）

`kernel/src/framebuffer.rs` が最下層の Graphics Backend に相当する部分の
最小実装です。

- `bootloader_api::info::FrameBuffer` からピクセルフォーマット
  (RGB/BGR/グレースケール) を判定し、抽象化して扱う
- `noto-sans-mono-bitmap` クレートでビットマップフォントをラスタライズし、
  背景色から前景色へのアルファブレンドでアンチエイリアス表示

## 今後のステップとの対応

- Step 11（Compositor / Window Manager の基礎）でダブルバッファリングと
  複数ウィンドウの合成を実装
- Step 12（Moon Shell）でステータスバー・ドック・ランチャーを実装
- Step 13（ウィジェットツールキット）でボタン・テキストボックス等の
  再利用可能な UI コンポーネントと Moon Light テーマエンジンを実装

GUI は既存 OS の見た目をそのまま模倣せず、上記のトークン（色・角丸半径・
アニメーション時間）を中心にオリジナルのビジュアル言語として設計します。
