# MonnOs Season 1

## Platforms
- iOS
- Android
- Windows
- macOS

## Core Gameplay
- Ride Scooter, Skateboard, and BMX.
- Independent physics for each discipline.
- Realistic rider balance and landing.
- Seamless equipment switching.

## World
- One open city with parks, plazas, rails, bowls, stairs and street spots.

## Systems
- Cross-platform save
- Progression
- Cosmetics
- Replay
- Photo Mode

## Development Order
1. Player Controller
2. Vehicle Physics
3. Trick System
4. Camera
5. Test Map
6. UI
7. Multiplayer foundation
